# Matrix Inc

**Opdracht KE03-INTDEV, ZUYD**

Een ASP.NET Core Razor Pages webapplicatie voor het verkopen van mechanische onderdelen.

## Project Starten

1. Open `MatrixInc.sln` in Visual Studio
2. Zet `MatrixInc.Web` als startup project (rechtsklik → Set as Startup Project)
3. Druk op **F5** om te starten
4. De database wordt **automatisch aangemaakt** en gevuld met seed data

## Database Setup

De database wordt **automatisch aangemaakt** bij de eerste start!

### Test Accounts

De database bevat 2 test klanten die je kunt gebruiken op de "Mijn bestellingen" pagina:
- `jan.jansen@example.com`
- `marie.peters@example.com`

## 📊 Database Bekijken

### In Visual Studio

1. Open **View → SQL Server Object Explorer**
2. Vouw uit: **SQL Server → (localdb)\mssqllocaldb → Databases → MatrixIncDb**
3. Bekijk de tabellen en data
